📊 Superstore Sales & Profit Analysis Dashboard
🚀 End-to-End Data Analytics Project using Power BI, SQL, and Excel
________________________________________
🔹 Project Overview
This project focuses on analysing retail sales data to uncover insights into profitability, regional performance, and product trends.
The dashboard is designed to help business stakeholders:
•	Monitor key performance metrics 
•	Identify loss-making areas 
•	Make data-driven decisions 
________________________________________
🔹 Tools & Technologies Used
•	Power BI – Data Visualisation & Dashboard Development 
•	SQL Server – Data Storage & Querying 
•	Excel – Data Cleaning & Initial Exploration 
•	Power Query – Data Transformation 
•	DAX – Calculated Measures & KPIs 
________________________________________
🔹 Key Features
•	📌 Interactive dashboard with slicers (Year, Month, Region) 
•	🔁 Toggle button (Sales ↔ Profit) using bookmarks 
•	🗺️ Filled map for regional performance analysis 
•	📊 KPI cards for Sales, Profit, Orders, and Profit Ratio 
•	📉 Matrix table with conditional formatting (profitability insights) 
•	🔍 Key Influencers & Decomposition Tree for root cause analysis 
•	🧭 Drill-down and cross-filtering across visuals 
________________________________________
🔹 Key Business Insights
•	💡 Technology category generates the highest sales 
•	⚠️ Tables sub-category consistently incurs losses 
•	📉 High discounts negatively impact profitability 
•	🌍 West region performs best in terms of revenue 
•	🚨 Certain products generate repeated losses and require attention 
________________________________________
🔹 Business Recommendations
•	Reduce discounts on loss-making products 
•	Focus on high-performing categories like Technology 
•	Improve pricing strategy in underperforming regions 
•	Optimise or discontinue loss-making products consistently
