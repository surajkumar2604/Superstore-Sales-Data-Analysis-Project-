📊 Superstore Sales Analysis Dashboard
🔹 Project Overview

This project analyzes retail sales data to uncover insights on profitability, regional performance, and product trends using Power BI.

🔹 Tools Used
Power BI (Dashboard & Visualization)
SQL Server (Data Storage & Queries)
Excel (Data Cleaning)
Power Query (Data Transformation)
🔹 Key Insights
Technology category generates the highest sales
Tables sub-category causes significant losses
High discounts reduce overall profitability
West region performs best in terms of sales
Some products generate consistent losses and should be reviewed
